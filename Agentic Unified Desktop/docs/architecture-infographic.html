Technical Deep Dive

The engineering decisions and technologies that make Athena reliable, fast, and secure.

API Layer

RESTful endpoints built on Node.js/Express. Stateless design with in-memory stores for demo simplicity. Supports both synchronous responses and SSE streaming.

Runtime Node.js 18+
Framework Express.js
Streaming SSE (EventSource)
üß†

Neuro¬Æ SAN Integration

Primary orchestration via Neuro¬Æ SAN semantic agent networks. Supports streaming chat mode for progressive token delivery. Falls back to Azure OpenAI/GPT-4o when configured.

Protocol HTTP + SSE
Timeout Configurable (env)
Networks Per-widget routing
üîó

MCP + Graph Execution

Model Context Protocol (MCP) for deterministic tool execution. Spawns stdio child process to m365-admin-mcp server. Executes Graph operations with Entra ID client credentials.

Protocol MCP (stdio)
Auth Client Credentials
Graph API v1.0
üì°

Real-Time Streaming

Server-Sent Events (SSE) push Customer 360 updates, agent replies, and insight refreshes to all connected clients. De-duplication via traceId.

Transport SSE (text/event-stream)
Persistence In-memory (demo)
Dedup Key traceId

Stack HTML/CSS/JS
